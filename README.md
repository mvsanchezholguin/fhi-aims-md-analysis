# FHI-aims Molecular Dynamics Analysis Toolkit

## Overview

This repository contains Python tools for post-processing Molecular Dynamics
simulations performed with FHI-aims.

## Features

- Trajectory extraction (.xyz)
- Atomic force extraction
- Extended XYZ generation for OVITO
- Thermodynamic analysis
- Nosé–Hoover diagnostics
- Statistical analysis
- Figure generation
- CSV export
